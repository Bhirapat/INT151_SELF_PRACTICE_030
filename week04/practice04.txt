forEach()

ใช้สำหรับวน loop แต่ละ element

ตัวอย่างนี้เช็คว่า string ขึ้นต้นด้วย "m" หรือไม่ → เก็บลง array ใหม่

map()

ใช้แปลงค่าใน array เดิม → array ใหม่

ตัวอย่าง: แปลง object employee → ชื่อเต็มตัวใหญ่ทั้งหมด

filter()

ใช้คัดเลือกค่าที่ตรงเงื่อนไขจาก array

ตัวอย่าง: เลือกเฉพาะชื่อที่มี "jai"

reduce()

ใช้รวมค่าใน array ให้เหลือค่าเดียว (accumulator)

ตัวอย่าง: รวมราคาสินค้า (price × amount) ทุกชิ้นในตะกร้า
