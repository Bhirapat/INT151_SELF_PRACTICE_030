การเปรียบเทียบ object–ต้องเช็คค่าแต่ละ property แทนที่จะเปรียบเทียบ object ตรง ๆ

การตรวจสอบ input–ก่อนคำนวณต้องตรวจสอบค่าที่รับเข้ามา (เช่น ห้ามเป็น 0 หรือค่าติดลบ)

Math.ceil()–ใช้สำหรับปัดเศษขึ้น เหมาะกับการคำนวณจำนวนหน้า (pagination)

การใช้ object ซ้อน (nested object)–เพื่อเก็บข้อมูลหลายรูปแบบในโครงสร้างเดียว

การเขียน class และการสร้าง object ด้วย new–เข้าใจการใช้ constructor และ property
